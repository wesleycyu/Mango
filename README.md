###About
- A Rotten Tomatoes clone that allows users can list & review movies.
- Images can be uploaded via the carrierwave gem.
- Users with admin status can view, manage, and even impersonate users.


###Technology
- Ruby on Rails
- HTML/CSS/Javascript
- jQuery/Bootstrap
- PostgreSQL
